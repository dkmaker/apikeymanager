op-keysync APT repository

Install with:

# 1. Add the GPG key
curl -fsSL https://dkmaker.github.io/apikeymanager/gpg.key | sudo gpg --dearmor -o /usr/share/keyrings/op-keysync.gpg

# 2. Add the repository
echo "deb [arch=all signed-by=/usr/share/keyrings/op-keysync.gpg] https://dkmaker.github.io/apikeymanager stable main"   | sudo tee /etc/apt/sources.list.d/op-keysync.list

# 3. Install
sudo apt update
sudo apt install op-keysync

Source: https://github.com/dkmaker/apikeymanager
